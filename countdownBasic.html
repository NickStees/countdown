jQuery Countdown Basics

This page demonstrates the very basics of the jQuery Countdown plugin. It contains the minimum requirements for using the plugin and can be used as the basis for your own experimentation.

For more detail see the documentation reference page.

Create a jQuery Doc Ready function to activate multiple instances

This jQuery function(see second doc ready in head) will scan your entire page for tags with the Class name "countDownTo" and look for a second Class that is a UNIX timestamp. This allows you to write a PHP web application that spits out something like August 20th 2055. Then when the page is loaded the jQuery function will convert these to Countdown's. Plus if it fails it will you will still have a human readable date. Also its not limited to PHP many languages have functions to output UNIX timestamps. -Enjoy!
